## The Problem

Every major global crisis left signals weeks before it made headlines.

- COVID-19 supply chain collapse — factories shutting in Wuhan were visible 18 days before WHO declared emergency
- Suez Canal blockage — weather + vessel traffic patterns were detectable hours before grounding
- Ukraine grain crisis — troop movements + wheat futures + fertilizer shortages were clustering 14 days before invasion

**Humans cannot connect these dots fast enough. KAIROS can.**

---

## What KAIROS Does

KAIROS is a real-time global crisis intelligence system with three layers:

**1. Signal Harvester** — monitors global news across 14 supply chain and geopolitical categories, clusters weak signals into risk themes using AI, and scores each cluster with a proprietary **Kairos Score™** (0–100)

**2. Ripple Engine** — a 100-node supply chain knowledge graph traversed with BFS. Any disruption event triggers a cascade simulation — hop by hop, with severity decay, time-to-impact estimates, and affected company mapping

**3. Kairos Foresight Index™** — a single global number representing the current stability of the world's supply chain system. Updates every 30 minutes. Think Dow Jones, but for global crisis risk.

## Knowledge Graph

The supply chain graph contains **100 nodes** and **180 weighted edges** encoding real-world dependencies.

| Type | Count | Examples |
|---|---|---|
| Countries | 25 | Taiwan, China, USA, Ukraine, Saudi Arabia, Iran |
| Industries | 15 | Semiconductors, Automotive, Pharma, Energy, Shipping |
| Commodities | 11 | Crude Oil, Natural Gas, Wheat, Lithium, Rare Earths |
| Ports | 7 | Shanghai, Singapore, Rotterdam, Los Angeles, Dubai |
| Trade Routes | 6 | Suez Canal, Strait of Hormuz, Malacca, Panama Canal |
| Companies | 26 | TSMC, ASML, Aramco, Maersk, Apple, Boeing, Pfizer |

Edge weights encode dependency strength. High-risk nodes (TSMC, Strait of Hormuz) amplify incoming severity during traversal.

## Getting Started

### Prerequisites

- Python 3.11+
- Node.js 18+
- Groq API key — free at [console.groq.com](https://console.groq.com)
- NewsAPI key — free at [newsapi.org](https://newsapi.org)

### Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# Fill in your API keys in .env
uvicorn app.main:app --reload --port 8000
```

Backend live at `http://localhost:8000`
Interactive API docs at `http://localhost:8000/docs`

### Frontend

```bash
cd frontend
npm install
cp .env.local.example .env.local
# Set NEXT_PUBLIC_API_URL=http://localhost:8000
npm run dev
```

Frontend live at `http://localhost:3000`

## API Reference

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/v1/analyze` | Analyze a real disruption event |
| `POST` | `/api/v1/simulate` | Run a hypothetical What-If simulation |
| `GET` | `/api/v1/signals` | Live signal clusters + Kairos Foresight Index |
| `GET` | `/api/v1/historical` | Historical validation events with ripple chains |
| `POST` | `/api/v1/report` | Generate PDF crisis brief |
| `GET` | `/health` | Backend health check |

### Example Request

```bash
curl -X POST http://localhost:8000/api/v1/analyze \
  -H "Content-Type: application/json" \
  -d '{"description": "Taiwan semiconductor production drops 30% due to earthquake"}'
```
